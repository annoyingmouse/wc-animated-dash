<wc-animated-dash/>

Demo here: https://codepen.io/annoyingmouse/pen/wvLWYmY

A simple animated dash component.

Installation

<script type="module" src="https://unpkg.com/@annoyingmouse/wc-animated-dash/wc-animated-dash.js"></script>

Alternatively:

<script type="module" src="https://cdn.skypack.dev/@annoyingmouse/wc-ace-spinner/wc-ace-spinner.js"></script>

Configuration

  • Adding a valid width attribute allow you change the size of the element - it defaults to 11rem wide, any valid CSS value will change this.
  • Adding a valid colour attribute with a value will allow you to set the colour of the dash
  • Adding a valid background-color attribute with a value will allow you to set the background colour
